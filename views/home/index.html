北京服装学院 2015 年特色资源库建设

CHARACTERISTIC RESOURCE LIBRARY SYSTEM

北京服装学院特色资源库 2014 年项目选取 3 个主题资源包进行建设。同时对原有的特色资源统一管理平台也需要进一步完善和推广，使其更好地为教学、科研服务。该项目的建设将有力地促进学校站在全局的高度，对全校特色资源进行统一规划、开发建设及整合，很好地推动了全校特色资源建设速度。通过进一步的开发建设，将使我校的特色资源更好地服务于教学、科研和社会。
